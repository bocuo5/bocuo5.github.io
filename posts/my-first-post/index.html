hugo主题配置

主题简单说明
技巧

简介：

主题名叫 hugo-theme-itheme ，是一款苹果味十足的Hugo主题

预览： https://hugo-theme-itheme.netlify.app

github中文介绍文档：

https://github.com/floyd-li/hugo-theme-itheme/blob/master/README_CN.md#site-configuration

hugo官网介绍：

https://themes.gohugo.io/themes/hugo-theme-itheme/

重点：

文章开头的设置 

title: 'Some Article' //标题名称
date: '2022-10-27T13:06:38+08:00'
draft: true //true 关闭；false 开启
description: 'some description of the article' //小标题
author: 'author of this article' //作者
cover: 'https://example.com/cover.jpg' // 封面图，如无配置，将用站点配置中的defaultCover
tags: ["tag1", "tag2", "tag3"] //文章标签
theme: "dark" // 设置文章的明暗主题(light/dark)

调整内文图片尺寸

文章内图片有三种显示模式，分别为

inline（中杯），big（大杯），wide（特大杯）

应用举例：

![中杯|inline](这里填入图片的链接)

分隔符是英文字符|，如果未设置则默认显示大杯big。
