Hugo基本操作

Hugo的记录
技巧

官方文档

https://gohugo.io/getting-started/quick-start/

重点

以下代码部分都是在PowerShell中输入

新增博客

PowerShell 内输入以下代码

hugo new posts/文件名字.md

新增的文件在 content/posts 目录下。使用编辑器打开文件即可编辑内容。

也可以先写好内容，然后把md文件放到该目录中

查看博客

开启本地调试，默认地址是：http://localhost:1313/。md 文档修改时，在 html 页面上可以实时看到变化；

hugo server

Ctrl + C 停止 Hugo 的开发服务器

运行以下任一命令可以预览草稿内容

hugo server --buildDrafts
hugo server -D
